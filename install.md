# bstack installed

The Broomva Stack is ready. Run `bstack status` to see your 16-skill inventory.

## Quick start

1. `bstack status` — see what's installed vs missing
2. `bstack bootstrap` — install all 16 skills
3. `bstack validate` — check health of every skill
